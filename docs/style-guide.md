# Curl Up & Dye Style Guide

## Color Palette
- Warm Cream: `#f7f2ea`
- Muted Stone: `#e8dfd2`
- Soft Peach-Beige: `#e7c8b5`
- Deep Ink: `#1e2533`
- Charcoal: `#32353d`

## Typography
- Headings: Playfair Display (fallback Georgia serif)
- Body: Inter (fallback system sans)

## Spacing Scale
- `--space-1` 0.5rem
- `--space-2` 0.75rem
- `--space-3` 1rem
- `--space-4` 1.5rem
- `--space-5` 2rem
- `--space-6` 3rem
- `--space-7` 4rem

## Components
- Primary button: `.btn.btn-primary`
- Secondary button: `.btn.btn-secondary`
- Content cards: `.card`
- Service blocks: `.service-block`
- Testimonial quote: `.quote`
- CTA band: `.cta-band`

## Dreamweaver Notes
- Global visual rules are in `css/styles.css`
- Keep page-specific content in each HTML file
- Repeated sections use matching class names for easy Find/Replace
